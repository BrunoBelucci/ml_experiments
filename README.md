# ml_experiments

**ml_experiments** is a Python package designed to orchestrate and manage machine learning experiments. It provides tools for organizing the code of machine learning experiments, automatically tracking parameters and metrics using ``mlflow``, preprocessing datasets, and performing HPO (Hyperparameter Optimization) using ``optuna``.

The basic usage of the package is illustraded in the notebooks ``base_usage.ipynb`` and ``hpo_usage.ipynb``.

# Installation

You can install the package using pip (inside the directory where pyproject.toml is located, as the package is not yet available trough PyPI):

```bash
pip install ml_experiments
```
